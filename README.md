# Educational Social Media Platform

## Description

This project is an **Educational Social Media Platform** developed using ASP.NET Core and Microsoft SQL Server. It is designed to support **students, teachers, and educational organizations** by enabling academic communication, content sharing, and collaborative learning within a structured educational environment.

Unlike general-purpose social media platforms, this system is **focused on education**, providing tools for:

* Students to interact, learn, and share knowledge
* Teachers to post lessons, resources, and assignments
* Educational organizations to manage academic content and communications

## Technologies Used

* ASP.NET Core
* Entity Framework (EF6)
* MS SQL Server 2014
* C#
* Visual Studio

## Key Features (Implemented)

* User registration and authentication
* Profiles for students, teachers, and organizations
* Posting, sharing, and commenting on educational content
* Role-based access control for different user types
* Secure storage and retrieval of academic data

## Planned Features

* Real-time notifications for academic updates
* Group or community features for collaborative learning
* Advanced content analytics and reporting
* Enhanced user interface for better academic interaction

## System Architecture

The platform follows a **layered architecture**:

* **ASP.NET Core** for application logic
* **Entity Framework** for data access
* Separation of concerns ensures maintainability and scalability

## Setup Requirements

* .NET SDK
* MS SQL Server 2014 or later
* Visual Studio or Visual Studio Code

## Purpose

This project was developed for **academic and learning purposes** and to explore the design and implementation of a **new concept in educational social media**. It demonstrates:

* Backend development skills
* Database integration
* System design for multi-user academic platforms
* Innovation in educational technology
